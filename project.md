# Weboldal tema 

Mongolia for traveling. Bemutatja Mongólia népszerű úti céljait, kultúráját és utazási tippeket ad turistáknak.

# programming language

- HTML
- CSS
- Javascipt

# Oldalak, funkciók rövid leírása

- **Főoldal**: Rövid bemutató Mongóliáról és látnivalóiról.
- **Úti célok oldal**: Ajánlott helyszínek bemutatása.
- **Kultúra oldal**: Információk a mongol hagyományokról és életmódról.

# nav bar 

Amikor rákattint egy navigációs barra, megjeleníti azt a szekciót és elrejti a többit

# javascript

- **TÖMB**

Egy lista, amely eltárolja az úti célokat, mint egy bevásárlólista.

- **FÜGGVÉNYEK**

Újrahasználható kódrészek, amelyek specifikus feladatokat látnak el, mint szerszámok egy szerszámosládában.

- **LOGIKAI MŰVELETEK**

Alapvető matematikai műveletek, mint összeadás és számlálás.

- **ELÁGAZÁS**

Ismétli a kódot minden tömbelemre, mintha egyesével végigmennénk az összes úti célon.

- **KERESÉS (Searching)**

1. Végigmegy minden úti célon
2. Ellenőrzi, hogy tartalmazza-e a keresett szöveget
3. Megjeleníti az összes találatot

- **SZÁMLÁLÁS (Counting)**

1. 0-ról indul a számlálás
2. Megnézi minden úti célt
3. Minden "tó" szót tartalmazó névnél növeli a számlálót 1-gyel
4. Visszaadja a végső számot

- **MAXIMUM KERESÉS**

1. Az első úti célt veszi leghosszabbként
2. Összehasonlítja minden úti cél nevének hosszát
3. Ha hosszabb nevet talál, az lesz az új leghosszabb
4. Végül megjeleníti a megtalált leghosszabb nevet
