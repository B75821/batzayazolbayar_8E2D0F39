# Weboldal tema 

Mongolia for traveling. Bemutatja Mongólia népszerű úti céljait, kultúráját és utazási tippeket ad turistáknak.

# programming language

- HTML
- CSS
- Javascipt

# Oldalak, funkciók rövid leírása

- **Főoldal**: Rövid bemutató Mongóliáról és látnivalóiról.
- **Úti célok oldal**: Ajánlott helyszínek bemutatása.
- **Kultúra oldal**: Információk a mongol hagyományokról és életmódról.
